Wenn die Breite einer Spalte geändert wird, dann wird es durch die letzte Spalte kompensiert. 
Wenn eine Spalte ausgeblendet wird, dann wird es durch die letzte Spalte kompensiert.
Wenn eine Spalte wieder eingeblendt wird, wird es durch alle Spalten kompensiert.
